Snake Omics technical report
============================

This is Snakemake's technical report for rule provenance and declared report
attachments. The separate reader-facing overview is generated at
``results/report/report.html`` by the ``report`` target. Use a different output
name, such as ``results/report/snakemake_report.html``, for this technical
report. Treat this technical report as internal/debug output unless it has
received a separate privacy review.

Large matrices and original images remain outside both HTML files and are
referenced through the artifact manifest.

The QC readiness score is a screening and review aid. It does not automatically
remove samples or spots, and missing evidence is never treated as a passing or
failing value.
